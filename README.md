# 2016级项目实训成果展示

## 明心心理咨询——大数据方向

### 项目简介

本项目为明心社工事务所的企业网站，客户为北京通州明心社工事务所。主要功能为提供网上在线预约业务、网上在线咨询业务，以简化事务所客户的预约流程，提高事务所的工作效率、更好地拓展业务。
另外，项目网站还提供一些附加功能：在线倾听、在线课程、心理文章等；同时融入大数据方面的知识，进行相关的推荐和预测。
